# Online Library Management System (Java + MySQL)

## Features
- Manage Books
- Manage Students
- Issue/Return Logs
- Simple Swing-based UI

## Technologies
- Java (Swing)
- MySQL
- JDBC

## Setup
1. Create a MySQL database: `library_db`
2. Configure credentials in `DBUtil.java`
3. Run `Main.java`
